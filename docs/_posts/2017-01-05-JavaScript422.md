外部ライブラリなしで、とにかく拙速に日付を書式化しようというだけです。  
  
年4桁、月と日は0パディングして２桁にします。  
  
****  
```js:
var formatDate = function(d) {
	return ''
	    + d.getFullYear()
	    + ('0' + (d.getMonth() + 1)).substr(-2, 2)
	    + ('0' + d.getDate()).substr(-2, 2);
};
```  
  
こういう簡単すぎるネタは如何なものかと考えるプライドを振り切っての投稿です。
